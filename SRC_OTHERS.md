# 原始碼詳細說明：輔助系統與模板 (SRC_OTHERS)

本文件詳細記錄 `command`, `storage`, `taskmanager` 與 `templates` 包下的檔案。這些檔案負責管理管理員指令、資料庫存儲、計時任務與數據結構定義。

---

## 📂 command.executor (管理員指令實作)
此包包含所有以 `/指令` 形式執行的管理員功能。

| 檔案名稱 | 詳細功能說明 |
| :--- | :--- |
| **`L1Adena.java`** | 給予目標玩家指定數量的金幣。 |
| **`L1CreateItem.java`** | 根據道具 ID 生成物品並放入管理員背包。 |
| **`L1Recall.java`** | 將特定的玩家強制傳送到管理員目前的位置（召喚玩家）。 |
| **`L1Move.java`** | 管理員自我傳送到指定的座標 (X, Y, MapID)。 |
| **`L1Invisible.java`** | 使管理員進入隱身狀態，玩家與怪物皆無法察覺。 |
| **`L1Level.java`** | 直接修改目標玩家的等級。 |
| **`L1Kick.java`** | 將目標玩家強制踢下線。 |
| **`L1BanIp.java`** | 封鎖特定 IP 位址，使其無法連線至伺服器。 |
| **`L1Speed.java`** | 賦予管理員或玩家加速狀態。 |
| **`L1Summon.java`** | 召喚指定的怪物到管理員身邊。 |
| **`L1SpawnCmd.java`** | 在目前位置永久增加一個 NPC/怪物的出生點。 |
| **`L1Status.java`** | 顯示或修改玩家的詳細屬性（力量、智力、血量等）。 |

---

## 📂 storage (資料庫存儲與持久化)
負責將運行中的物件數據（如玩家資訊、道具清單）保存至資料庫中。

| 檔案名稱 | 詳細功能說明 |
| :--- | :--- |
| **`mysql/MySqlCharacterStorage.java`** | 負責處理玩家角色數據（等級、座標、職業等）在 MySQL 資料庫的讀取與寫入。 |
| **`mysql/MySqlCharactersItemStorage.java`** | 負責處理玩家背包物品（強化等級、數量、屬性）在資料庫的保存與載入。 |
| **`CharacterStorage.java`** | 定義角色存儲的標準抽象介面。 |

---

## 📂 taskmanager (計時任務系統)
負責管理伺服器的後台定時任務（如：定時重啟、自動維護）。

| 檔案名稱 | 詳細功能說明 |
| :--- | :--- |
| **`TaskManager.java`** | 計時任務的核心管理器，負責調度所有後台 Job。 |
| **`tasks/TaskShutdown.java`** | 處理定時關閉伺服器的具體執行邏輯。 |
| **`tasks/TaskRestart.java`** | 處理定時重新啟動伺服器的具體執行邏輯。 |

---

## 📂 templates (數據模板/容器)
這些檔案是簡單的數據結構類別 (POJO)，對應資料庫中的每一張表格結構。

| 檔案名稱 | 詳細功能說明 |
| :--- | :--- |
| **`L1Item.java`** | 定義一個基準道具的所有屬性模板。 |
| **`L1Npc.java`** | 定義一個基準 NPC/怪物的屬性模板。 |
| **`L1Skills.java`** | 定義一個魔法技能的屬性模板。 |
| **`L1Armor.java`** | 特化用於防具類物品的模板。 |
| **`L1Weapon.java`** | 特化用於武器類物品的模板。 |
| **`L1Castle.java`** | 存儲城堡屬性（稅率、領地）的容器。 |
| **`L1SpawnTime.java`** | 存儲怪物出生時間限制條件的容器。 |

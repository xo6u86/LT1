# 原始碼詳細說明：遊戲物件模型 (SRC_MODEL)

本文件詳細記錄 `net.l1j.server.model` 及其子包內的所有類別。這是伺服器中最核心的部分，涵蓋了活體物件實體、戰鬥公式、遺產管理與世界邏輯。

---

## 📂 net.l1j.server.model (根目錄) 核心類別

| 檔案名稱 | 詳細功能說明 |
| :--- | :--- |
| **`L1Object.java`** | **萬物之源**。伺服器內所有物理存在的對象（玩家、NPC、道具、陷阱、甚至無形的標記點）的最頂層基類。 |
| **`L1Character.java`** | **活體基類**。繼承自 `L1Object`，是所有具備血量 (HP)、魔力 (MP) 與移動能力實體（玩家、怪物）的共同父類別。 |
| **`L1World.java`** | **世界管理器**。全伺服器在線物件的索引中心，負責快速查找玩家、NPC 或地上的物件，並處理全服廣播。 |
| **`L1Attack.java`** | **物理戰鬥核心**。處理近戰與遠端物理攻擊的所有計算，包含：命中判定、傷害數值公式、閃避計算、屬性加成與特效觸發。 |
| **`L1Magic.java`** | **魔法戰鬥核心**。處理所有魔法攻擊的機率判定、傷害計算、魔法抗性 (MR) 減免與特定異常狀態的施加。 |
| **`L1Inventory.java`** | **基礎容器類別**。管理物件容器（如背包、倉庫、怪物掉落包）內部的物品增刪查改。 |
| **`L1PcInventory.java`** | **玩家背包實作**。擴展自 `L1Inventory`，專門處理玩家背包的負量計算、裝備狀態更新與道具排序。 |
| **`L1Teleport.java`** | **傳送系統控制**。執行座標跳轉的核心邏輯，負責更新玩家的地圖座標並通知周圍玩家。 |
| **`L1HateList.java`** | **怪物仇恨管理**。AI 系統的核心，用於記錄與計算怪物對不同目標的「恨意值」，決定優先攻擊對象。 |
| **`L1Spawn.java`** | **出生控制邏輯**。根據設定檔的參數（地點、範圍、重生時間）將 NPC 或怪物實例化並放入世界地圖。 |
| **`L1War.java`** | **攻城戰核心系統**。處理攻城戰的宣告、城堡守護塔狀態檢測、血盟摸邊（皇冠）邏輯與獲勝城市交接。 |
| **`L1Clans.java`** | **血盟邏輯管理**。處理血盟創立、成員權限、封號設定以及血盟倉庫的使用限制。 |
| **`L1MapArea.java`** | **地圖區域定義**。用於界定一個矩形或特定形狀的座標範圍（如：安全區、戰鬥區、特定任務區）。 |
| **`AcceleratorChecker.java`** | **反加速器檢測**。分析封包發送的時間頻率，判斷玩家是否使用了惡意加速軟體。 |
| **`Hp/MpRegeneration`** | **生命/魔力回復**。計算角色每隔一段時間回復 HP/MP 的數值（受基礎數值、裝備、藥水、魔法娃娃影響）。 |
| **`L1PolyMorph.java`** | **變身系統**。處理玩家的外觀變換，包含移動速度、攻擊速度的同步調整與變身卷軸的時效。 |

## 📂 net.l1j.server.model.instance (實體包)
此包內的類別代表了遊戲中真實存在的「活體」或「互動對象」。

| 檔案名稱 | 詳細功能說明 |
| :--- | :--- |
| **`L1PcInstance.java`** | **玩家實體**。全專案**最核心**的檔案。代表一位在線玩家，封裝了其所有屬性、封包發送、動作行為與狀態切換。 |
| **`L1NpcInstance.java`** | **NPC 基類**。定義了非玩家角色的共通行為（如：對話、移動、受傷、消失）。 |
| **`L1MonsterInstance.java`** | **怪物實體**。負責怪物的 AI 尋路、主動攻擊判定、追蹤策略與死亡後的掉落物處裡。 |
| **`L1ItemInstance.java`** | **道具實體**。代表一個具體的物品（在背包內或地上），管理其強化等級、鑑定狀態、耐久度或數量。 |
| **`L1SummonInstance.java`** | **召喚獸實體**。受玩家魔法召喚的怪物，包含自動跟隨與攻擊目標的特殊 AI。 |
| **`L1PetInstance.java`** | **寵物實體**。被玩家馴服的動物，包含成長、飽食度、名字與裝備欄位管理。 |
| **`L1DollInstance.java`** | **魔法娃娃實體**。跟隨在玩家身邊提供 Buff 效果的特殊對象。 |
| **`L1MerchantInstance.java`** | **商店 NPC**。與玩家進行道具買賣、回收的核心實體。 |
| **`L1TeleporterInstance.java`** | **傳送師 NPC**。處理與玩家對話並開啟傳送清單的功能。 |
| **`L1DoorInstance.java`** | **門實體**。控制城堡大門、迷宮門的開啟、關閉與物理碰撞（損壞）。 |
| **`L1TowerInstance.java`** | **守護塔實體**。攻城戰中城堡核心區的標誌物，被破壞時會引發攻城邏輯切換。 |
| **`L1CrownInstance.java`** | **皇冠實體**。攻城戰結束後出現在地上的皇冠，盟主點擊即代表拿下城堡。 |
| **`L1EffectInstance.java`** | **特效實體**。地圖上的視覺特效或區域性傷害（如：火牆、毒霧）。 |
| **`L1DwarfInstance.java`** | **倉庫管理員**。提供玩家個人或血盟倉庫的存取對話。 |

## 📂 其餘功能子包 (Sub-packages)

### 1. model.item
- **`L1ItemDelay.java`**：管理所有道具的使用冷卻時間（如：喝藥水 500ms 延遲）。
- **`L1EquipmentSlot.java`**：處理裝備格位的邏輯（如：雙手武器不能拿盾、戒指數量限制）。

### 2. model.skill
- **`L1SkillUse.java`**：處理主動施展魔法的邏輯（檢查 Mp、材料、職業限制）。
- **`L1BuffUtil.java`**：輔助類別，用於快速給予玩家特定狀態效果。

### 3. model.map
- **`L1WorldMap.java`**：全局地圖管理。
- **`L1Map.java`**：單個地圖物件，處理座標的可行走判定 (Passable) 與座標查詢。

### 4. model.trap
- **`L1WorldTraps.java`**：管理散佈在世界各地的隱形陷阱。
- **`L1TrapInstance.java`**：單個陷阱的實例化行為（如：踩到會中毒或受傷）。

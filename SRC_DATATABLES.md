# 原始碼詳細說明：資料庫數據快取 (SRC_DATATABLES)

本文件詳細記錄 `net.l1j.server.datatables` 包下的所有類別。這些類別通常以 `getInstance()` 形式訪問，負責將資料庫中的靜態數據載入至記憶體以供遊戲邏輯快速調用。

---

## 📂 net.l1j.server.datatables 檔案清單

| 檔案名稱 | 詳細功能說明 |
| :--- | :--- |
| **`ArmorSetTable.java`** | **套裝屬性表**。定義完整的套裝（如：骨制套裝、神官套裝）及其額外獎勵屬性（如：力+2、回血增加）。 |
| **`AuctionBoardTable.java`** | **拍賣看板表**。負責讀取與更新拍賣場（如血盟小屋、特殊道具）的公告資訊與出價紀錄。 |
| **`BanNameTable.java`** | **禁字清單表**。存放系統禁止使用的角色名稱或血盟名稱關鍵字，用於防止不當命名。 |
| **`BoardTable.java`** | **公佈欄數據表**。管理遊戲內一般佈告欄 (BBS) 的文章內容、作者與發布時間。 |
| **`BossSpawnTable.java`** | **大怪出生表**。專門管理 Boss 級怪物的出現地點、重生間隔與隨機位置偏移。 |
| **`BuddyTable.java`** | **好友系統表**。處理玩家的好友清單讀取、新增與刪除，並同步到資料庫。 |
| **`CastleTable.java`** | **城堡數據表**。存放亞丁、肯特等各大城堡的所有權、稅率、金庫剩餘金額等資訊。 |
| **`CharBuffTable.java`** | **角色狀態暫存表**。負責在玩家下線時儲存當前的魔法狀態 (Buff)，並在重新上線時自動還原。 |
| **`CharacterConfigTable.java`** | **角色介面設定表**。儲存玩家自定義的快捷鍵設定、介面位置等個性化數據。 |
| **`CharacterTable.java`** | **角色基礎資料表**。負責讀取角色的基礎屬性（力量、敏捷、血量上限等）並處理非實體化狀態下的數據存取。 |
| **`ClanTable.java`** | **血盟數據表**。管理全伺服器所有血盟的資料，包含盟主、成員列表、盟標數據與血盟倉庫狀態。 |
| **`DoorSpawnTable.java`** | **門口/柵欄表**。管理地圖上所有門（如城堡大門、監獄門）的位置、初始狀態（開/關）與最大血量。 |
| **`DropItemTable.java`** | **掉落物基準表**。用於設定基礎的道具掉落規範（通常配合 DropTable 使用）。 |
| **`DropTable.java`** | **核心掉落表**。定義了每份怪物 (NpcId) 對應的掉落物清單、數量範圍與機率，是調整寶物難度的核心。 |
| **`ExpTable.java`** | **經驗值等級表**。定義了各等級所需的累計經驗值，並提供等級上限與萬能藥次數等參照數據。 |
| **`FurnitureSpawnTable.java`** | **家俱布置表**。記錄並還原玩家在血盟小屋內擺放的家俱位置與方向。 |
| **`GetBackRestartTable.java`** | **回村/重啟點表**。定義當玩家在特定地圖重啟或死亡後，系統應該將其傳送至哪一個地圖的哪一個座標。 |
| **`HouseTable.java`** | **小屋數據表**。管理亞丁、古魯丁等地所有玩家可租用或購買的小屋屬性、座標及其拍賣底價。 |
| **`IpTable.java`** | **IP 封鎖清單**。負責管理黑名單 IP，用於防火牆過濾與防止特定位址重複連線。 |
| **`ItemTable.java`** | **萬能道具總表**。遊戲中**最重要**的表之一，定義了所有道具（武器、防具、消耗品、任務道具）的原始屬性。 |
| **`LetterTable.java`** | **信件數據表**。記錄遊戲內玩家撰寫與寄送出的信件內容。 |
| **`LightSpawnTable.java`** | **靜態光源表**。定義地圖上裝飾性光源（如路燈、火把）的位置與光亮範圍。 |
| **`MailTable.java`** | **進階郵件表**。管理系統郵件、包裹等具有附件功能的複雜郵件數據。 |
| **`MapsTable.java`** | **地圖屬性總表**。定義每個 Map ID 的屬性，如：是否可傳送、是否為水域、是否有負重限制。 |
| **`MobGroupTable.java`** | **怪物群組表**。設定「怪物隊伍」，例如當白螞蟻出現時，身邊會隨機帶領幾隻兵蟻。 |
| **`MobSkillTable.java`** | **怪物技能對應表**。設定特定怪物 ID 可以使用的魔法技能（如：巴列斯會施展強力火球）。 |
| **`NPCTalkDataTable.java`** | **NPC 對話索引表**。將 NPC ID 與對應的 HTML 說明文檔（對話框）進行關聯。 |
| **`NpcActionTable.java`** | **對話觸發行為表**。定義點擊 NPC 對話框按鈕時，應該跳轉至哪一個 Java Handler 或 LScript。 |
| **`NpcChatTable.java`** | **NPC 閒聊台詞表**。存放 NPC 隨機說出的對話框文字。 |
| **`NpcSpawnTable.java`** | **靜態出生總表**。定義所有一般地圖上的 NPC 與怪物之座標、數量與重生間隔。 |
| **`NpcTable.java`** | **NPC 基本屬性表**。定義所有 NPC 的數值（HP、AC、力量、經驗值、GFX ID 等）。 |
| **`PetItemTable.java`** | **寵物道具表**。定義專供寵物裝備的防具或武器數據。 |
| **`PetTable.java`** | **寵物詳細資料表**。管理玩家已收服寵物的名稱、等級、目前屬性與飽食度。 |
| **`PetTypeTable.java`** | **寵物種類定義表**。定義不同寵物（小狼、杜賓狗等）的進化路徑與基礎潛力。 |
| **`PolyTable.java`** | **變身外觀表**。記錄所有變身代碼對應的 GFX 圖像與變身後的動作速度。 |
| **`ResolventTable.java`** | **溶解物品表**。定義使用「溶解劑」道具後，哪些裝備可以溶解成結晶體及其數量。 |
| **`ShopTable.java`** | **商店售價表**。管理所有 NPC 商店的進貨清單、售價以及回收價格。 |
| **`SkillsTable.java`** | **魔法技能定義表**。定義所有魔法的 Mp 消耗、屬性、傷害係數與效果持續時間。 |
| **`SpawnTable.java`** | **通用出生表**。一般用於特殊場景或舊版本兼容的怪物出生設定。 |
| **`SpawnTimeTable.java`** | **出生時間條件表**。與出生表配合，限定某些怪物只能在遊戲內的時間區段（如深夜）出現。 |
| **`SprTable.java`** | **圖形幀率表**。定義每個圖像編碼的播放速度與動畫間隔，用於同步客戶端顯示。 |
| **`TownTable.java`** | **村莊狀態表**。統計每個村莊的繁榮度、稅收盈餘與目前的領袖玩家。 |
| **`TrapTable.java`** | **地圖陷阱表**。定義隱形陷阱的位置與觸發效果（如：毒霧、受傷、傳送）。 |
| **`UBSpawnTable.java`** | **無限大戰怪表**。專門定義無限大戰 (Coliseum) 中每一波次出現的怪物清單。 |
| **`UBTable.java`** | **無限大戰 arena 表**。管理各個競技場的報名條件、限制人數與啟動時間。 |
| **`WeaponSkillTable.java`** | **武器額外效果表**。定義武器附帶的「發動」技能（如：毀滅者巨劍發動的吸血效果）。 |
